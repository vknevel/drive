Sharing

One-Step Sharing

Run nevel share on any file and get a clean web page back. A report reads like an article, a CSV becomes a table, a skill comes with an install button.

$ nevel share ./outputs/q2-review.md
scanning for secrets… clean ✓
published → https://share.navl.ai/q2-review
Q2 Review
share.navl.ai/q2-review

Instant Setup

Install Nevel and your team's skills, rules, and settings are already there. New hires run one sync and start where everyone else is.

$ nevel init
backing up ~/.claude … done ✓
pulling team skills … 5 installed ✓
installing rules, hooks, settings … done ✓
You're set up. Open Claude Code in any project.
